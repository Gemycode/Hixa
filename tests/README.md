# اختبارات API - Services Endpoints

هذا المجلد يحتوي على اختبارات للـ Services API endpoints.

## التثبيت

```bash
npm install
```

## تشغيل الاختبارات

```bash
# تشغيل جميع الاختبارات
npm test

# تشغيل الاختبارات في وضع watch mode
npm run test:watch
```

## الاختبارات المتوفرة

### Services Items
- ✅ إضافة خدمة جديدة (POST)
- ✅ جلب خدمة مع تفاصيلها (GET)
- ✅ تحديث خدمة (PUT)
- ✅ حذف خدمة مع تفاصيلها المرتبطة (DELETE)

### Services Details
- ✅ إضافة تفاصيل لخدمة محددة (POST)
- ✅ جلب تفاصيل خدمة محددة (GET)
- ✅ تحديث تفاصيل خدمة (PUT)
- ✅ حذف تفاصيل خدمة (DELETE)

## ملاحظات

- الاختبارات تستخدم MongoDB Memory Server (قاعدة بيانات مؤقتة في الذاكرة)
- لا تحتاج إلى قاعدة بيانات حقيقية للتشغيل
- جميع البيانات تُحذف تلقائياً بعد كل اختبار
